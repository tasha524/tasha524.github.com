CSS Cheat Sheet

Helpful CSS Link

1. color

Sets the color of the text.

color: red;

2. background-color

Changes the background color of an element.

background-color: blue;

3. margin

Creates space outside an element.

margin: 10px;

4. padding

Creates space inside an element between the content and the border.

padding: 10px;

5. border

Adds a border around an element.

border: 1px solid black;

6. width

Sets the width of an element.

width: 200px;

7. height

Sets the height of an element.

height: 100px;

8. display

Defines how an element is displayed on the page.

display: flex;

9. flex-direction

Controls the direction of items in a flex container.

flex-direction: row;

10. align-items

Aligns items vertically inside a flex container.

align-items: center;

11. justify-content

Controls how items are spaced horizontally in a flex container.

justify-content: space-between;

12. font-size

Changes the size of the text.

font-size: 20px;

13. text-align

Aligns text inside an element.

text-align: center;

14. border-radius

Rounds the corners of an element.

border-radius: 8px;

15. box-shadow

Adds a shadow around an element.

box-shadow: 0 0 10px gray;
