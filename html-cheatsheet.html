HTML Cheat Sheet

Helpful HTML Link

1. <h1>

Used for main headings on a page.

<h1>Heading</h1>

2. <p>

Used to display paragraphs of text.

<p>Paragraph</p>

3. <img>

Used to display an image on a webpage.

<img src="image.jpg">

4. <a>

Used to create a link to another page or website.

<a href="#">Link</a>

5. <ul>

Used to create an unordered list.

<ul></ul>

6. <table>

Used to create a table.

<table></table>

7. <tr>

Used to create a row inside a table.

<tr></tr>

8. <form>

Used to create a form which collects user input.

<form></form>

9. <input>

Used to create an input field in a form.

<input type="text">

10. <button>

Used to create a clickable button.

<button>Click</button>

11. <div>

Used as a container to group content.

<div></div>

12. <span>

Used to style or group inline text.

<span></span>

13. <link>

Used to connect an external CSS file.

<link rel="stylesheet">

14. <script>

Used to connect an external JS file.

<script></script>

15. <strong>

Used to emphasize a word.

<strong></strong>
